# Алхимик: ближайшие 3 итерации по химии

## Итерация 1. Школьное ядро и объясняющая теория
Цель:
- перестать показывать тонкие theory-заглушки,
- дать школьную базовую химию с объяснениями и примерами.

Состав:
1. Расширить темы 7-11 классов.
2. Для каждой темы добавить:
- explanation
- examples
- key terms
- formula block
- common mistakes
- mini-check
3. Довести экран темы до более понятной структуры.

Готовность итерации:
1. темы 8 и 9 класса уже воспринимаются как учебные, а не как краткие заметки;
2. есть примеры и объяснения;
3. пользователь понимает, что именно повторить.

## Итерация 2. Визуализация, практика, проверка
Цель:
- связать теорию с наглядностью и задачами.

Состав:
1. Иллюстрации и схемы по ключевым темам.
2. Анимации по выбранным процессам.
3. Новый task-flow по теме.
4. Разбор ошибок и рекомендации после тестирования.
5. Мини-проверочные по теме.

Готовность итерации:
1. теория -> визуализация -> практика связаны,
2. пользователь получает путь к повторению темы.

## Итерация 3. Учебные линии, ИИ-персонализация, экзамены
Цель:
- довести химию до реального personalized learning модуля.

Состав:
1. Две полноценные учебные линии как proof-of-concept.
2. AI-анализ ошибок и трудных тем.
3. Рекомендации повторить тему или задать вопрос AI.
4. Режим теста на время.
5. Демо ОГЭ / ЕГЭ / МЦКО по химии.
6. Банк задач и викторин без быстрого повторения.

Готовность итерации:
1. система не просто показывает ошибку, а предлагает маршрут закрытия пробела;
2. chemistry становится персонализированным учебным модулем.
