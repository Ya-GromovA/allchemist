# Алхимик: подробный backlog наполнения по химии

## 1. Цель
Системно наполнить химию так, чтобы она стала самым сильным модулем приложения.

## 2. Очередность наполнения
### Приоритет P0
1. Школьная химия 7-11 классы: базовая теория
2. Школьная химия 7-11 классы: теория по учебным линиям
3. Базовые задачи и проверочные по каждой теме
4. Визуализация ключевых школьных тем

### Приоритет P1
1. Демо ОГЭ по химии
2. Демо ЕГЭ по химии
3. Демо МЦКО по химии
4. Викторины по школьным темам

### Приоритет P2
1. Аналитическая химия
2. Физколлоидная химия
3. Биохимия
4. Углубленная органическая химия
5. Углубленная неорганическая химия

## 4. Формат наполнения по каждой теме
Для каждой темы нужны:
1. Краткая теория
2. Полная теория
3. Ключевые термины
4. Формулы и обозначения
5. Частые ошибки
6. Мини-проверка
7. Иллюстрации
8. Схемы
9. Анимации при необходимости
10. Задачи 3 уровней
11. Проверочная работа
12. Экзаменационные привязки
13. Parent note
14. Teacher note

## 5. Теория по учебным линиям
Нужно минимум начать с 2 proof-of-concept линий:
1. Габриелян
2. Рудзитис / Фельдман

Для каждой:
1. карта тем
2. порядок тем
3. mapping к параграфам
4. оригинальный пересказ
5. legal metadata

## 7. DoD по химии
Тема считается готовой только если:
1. есть теория
2. есть практика
3. есть проверка
4. есть хотя бы один визуальный блок
5. есть подтвержденный источник
