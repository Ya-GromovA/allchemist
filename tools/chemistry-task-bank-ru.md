# Алхимик: структура большого банка задач по химии

## 1. Цель
Сформировать большой, пополняемый, слабо повторяющийся банк задач, тестов и викторин по химии.

## 2. Базовые типы единиц банка
1. Теоретический вопрос
2. Тест с выбором ответа
3. Тест с несколькими правильными ответами
4. Короткий открытый ответ
5. Задача на расчет
6. Задача на уравнение реакции
7. Задача на установление соответствия
8. Лабораторный сценарий
9. Экзаменационное задание
10. Викторинный быстрый вопрос

## 3. Обязательные поля для каждой задачи
1. `id`
2. `subject = chemistry`
3. `grade`
4. `track`
- school
- exam
- advanced
5. `branch`
- inorganic
- organic
- analytical
- physical_colloid
- biochemistry
6. `topic_id`
7. `difficulty`
- basic
- standard
- advanced
8. `task_type`
9. `question`
10. `answer`
11. `explanation`
12. `mistake_tags`
13. `timed_expected_seconds`
14. `source_type`
15. `verified`

## 4. Разрезы банка задач
### По уровню
1. Школьный
2. Экзаменационный
3. Углубленный

### По цели
1. Проверка знаний
2. Тренировка
3. Повтор темы
4. Исправление конкретной ошибки
5. Викторина

### По скорости ответа
1. Без ограничения времени
2. Timed mode
3. Speed drill

## 5. Связь с AI-персонализацией
После теста система должна выбирать задания по:
1. темам с ошибками,
2. долгому времени ответа,
3. повторяющимся mistake tags,
4. уровню сложности пользователя.

## 6. Антиповторы
Чтобы задания не повторялись слишком часто, нужно хранить:
1. последние выданные id,
2. частоту показа,
3. процент совпадения по теме,
4. историю ошибок пользователя.

Правило:
- не выдавать подряд один и тот же вопрос,
- не собирать все тесты из одного микросета,
- балансировать темы и типы задач.

## 7. Минимальный стартовый объем
### Итерация 1
1. 7 класс: 80-120 задач
2. 8 класс: 180-220 задач
3. 9 класс: 180-220 задач
4. 10 класс: 180-220 задач
5. 11 класс: 180-220 задач

### Итого старт
Минимум 800-1000 школьных chemistry-задач.

## 8. Экзаменационный слой
Отдельно нужны:
1. ОГЭ chemistry set
2. ЕГЭ chemistry set
3. МЦКО chemistry set

Для каждого:
1. формат задания
2. критерий проверки
3. разбор
4. привязка к темам

## 9. Викторинный слой
Нужны короткие наборы:
1. 10-секундные вопросы
2. блиц-формат
3. парные сравнения
4. кто быстрее уравняет реакцию

## 10. DoD
Банк задач можно считать реально запущенным, если:
1. есть минимум 800+ школьных задач,
2. есть привязка к темам,
3. есть explanation для ответа,
4. есть mistake tags,
5. AI может выбирать задания для повторения, а не случайно.
