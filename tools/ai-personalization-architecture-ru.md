# Алхимик: AI-персонализация, закрытие пробелов и большой банк заданий

## 1. Цель
Алхимик должен помогать ученику и студенту не просто читать контент, а закрывать реальные пробелы в знаниях.

## 2. Что должен делать AI-сервис
1. Анализировать результаты тестов и задач.
2. Определять трудные темы.
3. Отмечать ошибки по типам.
4. Учитывать время ответа в timed mode.
5. Предлагать:
- повтор темы,
- короткую практику,
- вопрос AI,
- повторное тестирование.

## 3. Базовый цикл персонализации
1. Пользователь проходит тест.
2. Система видит ошибки и сложные места.
3. AI показывает результат понятным языком.
4. Предлагает темы для повторения.
5. После повторения предлагает пройти тест снова.

## 4. Что нужно хранить
1. тема ошибки
2. тип ошибки
3. время ответа
4. уверенность пользователя
5. сколько раз тема уже провалена
6. какие рекомендации уже были даны

## 5. Что показывать пользователю
1. У вас ошибки по темам...
2. Лучше повторить...
3. Вы дольше всего думали над...
4. Хотите, я объясню это заново?
5. После повторения пройдите мини-тест для закрепления.

## 6. Большой банк заданий
Нужны:
1. задачи по темам,
2. задачи по типам ошибок,
3. timed tests,
4. викторины,
5. обновляемые наборы вопросов,
6. стратегия уменьшения повторов одних и тех же задач.

## 7. Следующий этап реализации
После укрепления школьного chemistry core эта архитектура должна быть встроена:
1. в chemistry,
2. затем в physics,
3. затем в biology.
