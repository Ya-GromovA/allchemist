Руководство пользователя веб-версии Алхимик
Версия: 2026-02-26

1) Вход и роль
- Откройте веб-приложение Алхимик.
- Выберите роль: Ученик/Студент, Преподаватель, Родитель.
- Подтвердите профиль.

2) Что доступно сейчас
- Role-first web workspace.
- Ссылки: Web Admin и API Docs.
- Trust-блок с источником и датой обновления.

3) Типовые сценарии
- Ученик/Студент: урок -> практика -> прогресс.
- Преподаватель: тема -> задание -> контроль группы.
- Родитель: прогресс -> план -> контроль пробелов.

4) Проблемы
- Старая версия: Ctrl+F5.
- Нет доступа: проверьте роль и авторизацию.
- Не открываются ссылки: проверьте сервер.

5) Безопасность
- Используйте только доверенные URL Алхимик.
- Не передавайте учетные данные третьим лицам.

6) Поддержка
- Используйте официальный канал поддержки.
