Руководство пользователя мобильного приложения Алхимик
Версия: 2026-02-26

1) Начало работы
- Установите APK из официальной ссылки Алхимик.
- Запустите приложение и выберите роль: Ученик/Студент, Преподаватель, Родитель.
- Пройдите синхронизацию по телефону (опционально).

2) Основные действия
- Изучите тему в Уроках.
- Решите практику в Задачах.
- Отслеживайте результат в Аналитике и Кабинете.
- Используйте AI Наставник для разбора ошибок.

3) Смена устройства
- На старом устройстве: Синхронизировать аккаунт.
- На новом: подтвердить телефон и код.

4) Режимы сети
- Online: синхронизация и обновления доступны.
- Offline: работает локальный контент.

5) Типовые проблемы
- Нет запуска: перезапустите приложение и сеть.
- Нет обновлений контента: нажмите Проверить обновления контента.
- Нет SMS кода: проверьте номер и запросите код повторно.

6) Безопасность
- Не передавайте коды третьим лицам.
- Используйте только официальные сборки.

7) Поддержка
- Используйте официальный канал поддержки вашей организации.
