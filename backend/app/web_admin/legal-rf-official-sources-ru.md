# Официальные источники для юридического блока РФ

Версия: 2026-02-26

Ниже список официальных ресурсов, на которые стоит опираться при закрытии юридического блока.

1. Официальный интернет-портал правовой информации (pravo.gov.ru)
- https://pravo.gov.ru
- Актуальные тексты НПА и официальные публикации.

2. Официальное опубликование правовых актов (publication.pravo.gov.ru)
- https://publication.pravo.gov.ru
- Проверка факта и даты официального опубликования.

3. Роскомнадзор: персональные данные
- https://pd.rkn.gov.ru
- Требования по ПДн, реестр операторов, методические материалы.

4. ФСТЭК России
- https://fstec.ru
- Требования и документы по защите информации в ИСПДн (по применимости).

5. ФСБ России
- http://www.fsb.ru
- Нормативная база по криптографической защите (если используется СКЗИ).

6. Банк России (если есть платежные сервисы)
- https://www.cbr.ru
- Разъяснения и требования для финансового контура (по применимости).

7. Роспатент (если важны товарные знаки и ПО)
- https://rospatent.gov.ru
- Регистрация ИС, проверка прав на бренд и ПО.

Важно:
- Этот список не заменяет юридическое заключение.
- Финальную конфигурацию документов и процессов должен утвердить профильный юрист по праву РФ.
