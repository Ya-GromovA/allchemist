# Руководство пользователя веб-версии Алхимик

Версия: 2026-02-26

## 1. Вход и роль
- Откройте веб-приложение Алхимик.
- На стартовом экране выберите роль: Ученик/Студент, Преподаватель, Родитель.
- Подтвердите профиль, чтобы получить персональный стартовый сценарий.

## 2. Что доступно сейчас
- Role-first веб-рабочее пространство.
- Быстрые операционные ссылки (Web Admin, API Docs).
- Trust-блок с прозрачностью источника и даты обновления.

## 3. Типовые сценарии
- Ученик/Студент: урок -> практика -> проверка прогресса.
- Преподаватель: подготовка темы -> выдача задания -> контроль группы.
- Родитель: просмотр прогресса -> план на день -> контроль пробелов.

## 4. Типовые проблемы
- Видите старую версию экрана: Ctrl+F5.
- Нет доступа к разделу: проверьте роль и авторизацию.
- Не открываются ссылки: проверьте доступность сервера.

## 5. Безопасность
- Используйте только доверенные URL Алхимик.
- Не передавайте учетные данные третьим лицам.

## 6. Поддержка
- Обращайтесь по внутреннему каналу поддержки вашей организации.
