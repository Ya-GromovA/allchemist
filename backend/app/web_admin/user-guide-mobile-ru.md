# Руководство пользователя мобильного приложения Алхимик

Версия: 2026-02-26

## 1. Начало работы
- Установите APK из официальной ссылки Алхимик.
- Запустите приложение и выберите роль: Ученик/Студент, Преподаватель, Родитель.
- Пройдите шаг синхронизации по телефону (по желанию, для переноса доступа на другое устройство).

## 2. Основные действия
- Откройте экран Уроки и изучите тему.
- Перейдите в экран Задачи и решите практический блок.
- Проверяйте прогресс в Аналитике и Кабинете.
- В AI Наставнике задавайте вопросы по теме и просите разбор ошибок.

## 3. Смена устройства
- На старом устройстве нажмите Синхронизировать аккаунт.
- На новом устройстве в onboarding подтвердите телефон и код.
- Проверьте, что доступные модули и настройки восстановились.

## 4. Режимы сети
- Online: доступна синхронизация и обновление контента.
- Offline: открывается локальный контент; действия, требующие сервера, недоступны.

## 5. Типовые проблемы
- Приложение не загружается: перезапустите приложение и сеть.
- Нет новых паков контента: нажмите Проверить обновления контента.
- Не приходит код телефона: проверьте формат номера и повторите запрос.

## 6. Безопасность
- Не передавайте одноразовые коды и доступ к устройству третьим лицам.
- Используйте только официальные сборки приложения.

## 7. Поддержка
- Обратитесь в поддержку через назначенный канал вашей организации.
