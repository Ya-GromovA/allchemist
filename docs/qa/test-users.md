# Тестовые пользователи Allchemist

Единый пароль для всех тестовых аккаунтов: `AlchTest2070`.

Эти пользователи нужны для QA web, mobile, admin/API и проверки ролевой видимости. Токены не сохраняются в документации.

Короткие QA-логины для Android-эмулятора используют те же backend-пользователи, роли, пароли и доступы:

- `s2070` -> `alch_test_student_school`;
- `t2070` -> `alch_test_teacher_school`;
- `h2070` -> `alch_test_homeroom_school`;
- `p2070` -> `alch_test_parent_school`.

## Проверено автоматически

- Production health: `https://api.allchemist.ru/api/v1/health`.
- Вход через `/api/v1/auth/login`.
- Проверка роли через `/api/v1/auth/me` и `/api/v1/users/profile`.
- Проверка активного доступа через `/api/v1/users/access`.

## Ручная проверка web

1. Открыть `https://api.allchemist.ru/api/v1/web`.
2. Выбрать сценарий входа, подходящий роли.
3. Ввести логин и пароль `AlchTest2070`.
4. Проверить, что заголовок и рабочие разделы соответствуют роли.
5. Проверить, что технические ключи и английские статусы не отображаются пользователю.

## Ручная проверка mobile

1. Установить актуальный APK из `/home/usgromov/Allchemist/apk/`.
2. Выбрать сценарий входа.
3. Войти логином и паролем `AlchTest2070`.
4. Проверить нижнее меню и первую главную страницу после входа.
